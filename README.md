# drf_test


# What is it

Test task for company

# Requirements

- Flask
- SQLAlchemy
- Flask-Admin

# How to use guide

1. Install dependencies
```
$ pip install -r requirements.txt
```
2. Make database migrations 
```
$ python manage.py runserver
```

3. Generate test data using python script
```
$ python 
```

4. run django app
```
$ python manage.py runserver
```
